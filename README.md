# 什么什么？我买的东西更贵？
价格歧视、大数据杀熟的简单模拟 - OUC 25秋ai与数字经济学

## 项目介绍
基于Streamlit构建的交互式网页应用，用于模拟淘宝/京东等平台的"千人千价"现象。用户可以：
- 输入个人特征（用户类型、消费水平、设备等）
- 查看个性化定价结果
- 了解背后的经济学原理
- 学习应对策略

## 功能特点
- [x] 个性化价格计算  
- [x] 价格构成明细展示  
- [x] 经济学原理解释  
- [x] 群体价格分布模拟s  
- [x] 应对策略建议  

## 在线体验
部署链接：[Streamlit](https://taobao-price-simulator.streamlit.app)

## 本地运行

### 1. 克隆项目
```bash
  git clone https://github.com/aiCane/taobao-price-simulator.git
```

### 2. 安装依赖
```bash
  pip install -r requirements.txt
```

### 3. 运行应用
```bash
  streamlit run app.py
```
